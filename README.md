# GrowthLedger

Smart To-Do List with daily growth analysis. Built with React + Vite. Ready for Vercel.

Quick start:

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Notes:
- Uses `localStorage` to persist lists and archives.
- Charts powered by `chart.js` and `react-chartjs-2`.
- Toggle day/night mode via header button.
